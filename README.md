# onuxt-ionic

```bash
pnpm i
```

Open the Ionic VSCode Extension, from your sidebar  -> Settings -> Activate Live Reload.

Then, Run -> Android/iOS.

That's it ! Enjoy Live Reload on your physical device / emulator / simulator.
